Depois de alguns anos usando um template super simples do VuePress, resolvi recriar meu site com foco nas minhas streams na [Twitch](https://twitch.tv/mathewcst).

Site feito com [Nuxt.js](https://nuxtjs.org/) em modo `static`. O layout pode ser encontrado [aqui](https://www.figma.com/file/dgFn8CFKA8FWZ6MG64z99VfV/MathewCosta.com?node-id=55%3A2).

![screenshot](https://353a23c500dde3b2ad58-c49fe7e7355d384845270f4a7a0a7aa1.ssl.cf2.rackcdn.com/5f134346ba91d80007d6c827/screenshot.png)

## Rodando o Projeto

```bash
# instalar dependências
$ yarn install

# rodando localmente com hotreload em localhost:3000
$ yarn dev

# build para produção
$ yarn build
$ yarn start

# gerar site estático
$ yarn generate
```
